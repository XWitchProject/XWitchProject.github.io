XWitch Project

Set of libraries to help you develop external tools for Noita.

Resources

  • XWitch - (documentation)

    [C#] Schema specification library made to aid in understanding Noita's XML files

  • Schemas

    [Data] Specifications for various Noita files (materials.xml, data/schemas etc.) in XWitch format

  • Relua - (documentation)

    [C#] Parser and scripted transformation engine for Lua scripts

  • NXML - (documentation)

    [C#] Parser and scripted transformation engine for Noita's XML files
